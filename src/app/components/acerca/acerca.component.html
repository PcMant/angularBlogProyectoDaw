Acerca De

Esta aplicación ha sido desarrollada por Juan Molina Gómez como proyecto final del FP GS Desarrollo de Aplicaciones web.

Se tata de un sistema de blog que se compone de las siguientes tecnologías:
  • Base de datos MySQL.
  • Api RESTfull dinamica con PHP.
  • Angular como frontend.
  • Interfaz web con Bootstrap, SASS y HTML.
